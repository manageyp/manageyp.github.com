本篇记录了，通过在项目中引入 NewRelic ，监控 Rails 项目，以及 Grape API 的性能数据。

引入两个 Gem

Gemfile 

    # New Relic Ruby Agent
    gem 'newrelic_rpm', '3.9.6.257'
    gem 'newrelic-grape', '2.0.0'

创建配置文件，配置 Key

config/newrelic.yml 

    license_key: 'LICENSE KEY '

程序初始化

config.ru 

  # You need to manually start NewRelic agent
  NewRelic::Agent.manual_start
